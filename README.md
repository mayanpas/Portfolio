# 🚀 Portfólio Profissional | Mayan Patrick

Portfólio pessoal desenvolvido com **React**, focado em performance, design moderno, responsividade e experiência de usuário fluida (especialmente otimizado para dispositivos móveis e iOS).

---

## 🛠️ Tecnologias e Ferramentas

O projeto foi construído utilizando tecnologias modernas de desenvolvimento front-end:

* **React** — Biblioteca principal para construção da interface baseada em componentes.
* **Vite** — Empacotador e ferramenta de build de altíssima performance.
* **HTML5 & CSS3** — Estruturação e estilização avançada com foco em responsividade e animações fluidas.
* **Git & GitHub** — Versionamento de código.
* **Vercel** — Hospedagem e *Continuous Deployment (CD)*.

---

## ✨ Principais Funcionalidades

* **Design Responsivo & Adaptativo:** Layout adaptado para diferentes tamanhos de tela, com correções específicas de viewport para dispositivos móveis (`viewport-fit=cover`).
* **Navegação Inteligente:** Barra de navegação dinâmica com comportamento otimizado ao rolar a página e ao interagir com links internos.
* **Seções Organizadas:**
  * **Sobre Mim:** Apresentação profissional e objetivos de carreira.
  * **Habilidades:** Stack técnica detalhada.
  * **Projetos:** Vitrine de projetos desenvolvidos (com links para repositórios e demonstrações).
  * **Fale Comigo:** Canais de contato direto e redes sociais.

## 📂 Estrutura do Projeto

```text
Portfolio/
├── public/            # Arquivos estáticos e ativos públicos
├── src/
│   ├── components/    # Componentes modulares (Navbar, Seções, etc.)
│   ├── App.jsx        # Componente raiz da aplicação
│   ├── App.css        # Estilos globais e específicos da aplicação
│   └── main.jsx       # Ponto de entrada do React
├── index.html         # Arquivo HTML principal com meta tags de viewport
├── package.json       # Dependências e scripts do projeto
└── vite.config.js     # Configuração do Vite
```
